# Vim mini cheat sheet

#### Exist Vim

```
:q! - to trash all changes
:wq - to save the changes
```

#### Select
```
v - select characters
V - select whole lines
```

#### Delete
```
x  - delete the character at the cursor
d  - delete/cut
dd - delete/cut the whole line
```

#### Copy / Paste
```
y  - copy
yy - copy the whole line
p  - paste
```
